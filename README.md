# CMU Manager

Application web Django de gestion des cartes CMU.

## Objectifs
- Rechercher rapidement un client
- Afficher clairement son code de rangement
- Importer des fichiers Excel/CSV
- Gérer les doublons
- Gérer les utilisateurs selon des rôles

## Stack technique
- Django
- Django Templates
- Bootstrap 5
- SQLite
